管理者向けトラブル対応表
症状 原因・対処
「このメールアドレスで登録されたアカウントが見つかりません」 「新規登録」タブで登録していないまま「ログイン」タブを使っている。新規登録から始めるよう案内。
OTPが届かない（5分以上） 迷惑メール確認→「再送信」を試させる。それでもダメなら管理者が「👨‍🏫 生徒管理」でアカウントを確認し、必要なら削除して再登録させる。
「招待枠の上限に達しています」 「👨‍🏫 生徒管理」→該当生徒を選択→「枠数を変更」で上限を増やすか、既存のスロットを削除する。
チェックインが取り消したい 「🎫 ゲスト一覧」→該当スロットの「取消」ボタンで入場済みフラグをリセットできる。
管理者パスワードを忘れた サーバーに直接アクセスし、DBの users テーブルをリセット。詳細は system_manual.md の「エラー5」参照。
