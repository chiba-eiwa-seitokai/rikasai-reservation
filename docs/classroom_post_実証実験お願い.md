## 投稿本文（このまま貼り付け）

**件名：【生徒会】入場システム 実証実験へのご協力のお願い（6/19 金・登校時）**

梨花祭2026に向けて、生徒会では入場管理システム（QRコードで登校・入場をチェックインする仕組み）を開発しました。
本番（7/17〜18）で安定して使えるよう、**6月19日（金）の登校時に、実際にQRをかざすリハーサル（実証実験）** を行います。
つきましては、参加クラスのみなさんに以下のご協力をお願いします。

■ やってほしいこと（6月18日・木 までに準備）
1. スマホのブラウザで → rikasai-reservation.vercel.app/student を開き「新規登録」
2. 学校メール（@biblos.ac.jp）・氏名・クラスを入力し認証コードを受信→ログイン
3. ダッシュボードで「ゲストの名称」に **自分の名前** を入力→「招待リンクを生成」
4. 表示されたQRコードを **スクリーンショットで保存**（当日まで消さない）

■ 当日（6月19日・金 登校時）
昇降口でスマホのQRを係の端末にかざすだけ（数秒）。「ピッ」と鳴れば完了です。

■ 詳しい手順
rikasai-reservation.vercel.app/experiment に画像付きの手順があります。

■ 困ったとき
・QRを保存し忘れた → 当日もう一度ログインして手順3〜4をやり直せばOK
・メールが届かない → 迷惑メールフォルダを確認、それでも届かなければ生徒会へ
・スマホがない/忘れた → 係が名前で手動確認します

ご協力よろしくお願いします。
千葉英和高校 生徒会
