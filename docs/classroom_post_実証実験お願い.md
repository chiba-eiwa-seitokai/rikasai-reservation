## 投稿本文（このまま貼り付け）

**件名：【生徒会】入場システム 実証実験へのご協力のお願い（6/19 金・登校時）**

全校生徒のみなさんへ。

■ なぜこのシステムを作ったのか
梨花祭では例年、ゲストの来場者を紙の名簿で管理していましたが、混雑時の確認に時間がかかるという課題がありました。そこで生徒会では今年から、**スマホのQRコードをかざすだけで瞬時に入場チェックができる「入場管理システム」** を独自に開発しました。ゲスト1人あたりの受付を5秒以内に短縮し、梨花祭をもっとスムーズに楽しんでもらうことが目標です。

■ なぜ実証実験をするのか
システムの開発は完了していますが、まだ実際の運用経験がありません。本番（7/17〜18）でいきなり使うとトラブルのリスクがあるため、**事前に全校生徒で実際に使ってみるリハーサル** を行います。これにより、

- システムが大人数で安定して動くかを検証できる
- 係（生徒会役員）がQRスキャン操作に慣れられる
- 問題があれば本番前に修正できる

みなさんの協力が、梨花祭本番の成功に直結します。

梨花祭2026に向けて、**6月19日（金）の登校時に、実際にQRをかざすリハーサル（実証実験）** を行います。
つきましては、全校生徒のみなさんに以下のご協力をお願いします。

■ やってほしいこと（6月18日・木 までに準備）
1. スマホのブラウザで → rikasai-reservation.vercel.app/student を開き「新規登録」
2. 学校メール（@biblos.ac.jp）・氏名・クラスを入力し認証コードを受信→ログイン
3. ダッシュボードで「ゲストの名称」に **自分の名前** を入力→「招待リンクを生成」
4. 表示されたQRコードを **スクリーンショットで保存**（当日まで消さない）

■ 当日（6月19日・金 登校時 7:45〜8:35）
昇降口でスマホのQRを係の端末にかざすだけ（数秒）。「ピッ」と鳴れば完了です。
※ この時間帯に登校したら昇降口で係（生徒会役員）にQRを提示してください。

■ 詳しい手順
rikasai-reservation.vercel.app/experiment に画像付きの手順があります。

■ 困ったとき
・QRを保存し忘れた → 当日もう一度ログインして手順3〜4をやり直せばOK
・メールが届かない → 迷惑メールフォルダを確認、それでも届かなければ生徒会へ
・スマホがない/忘れた → 係が名前で手動確認します

ご協力よろしくお願いします。
千葉英和高校 生徒会
